在这个文件里调：`D:\My_Project\ski\frontend\src\components\SkiScen
   e3D.vue`。

主要亮度参数在这些位置：
• renderer.toneMappingExposure =
1.2：整体曝光，最直接，建议先调到 1.5 ~ 2.0
• const ambient = new THREE.AmbientLight(0xb0c4de,
0.6)：环境光强度，建议调到 0.9 ~ 1.2
• const sun = new THREE.DirectionalLight(0xfff5e0,
1.4)：主光源强度，建议调到 1.8 ~ 2.5
• const fill = new THREE.DirectionalLight(0x8ab4f8,
0.4)：补光强度，建议调到 0.7 ~ 1.0
• scene.fog = new THREE.FogExp2(0x0a1929,
0.008)：雾太重也会显暗，可把 0.008 降到 0.003 ~ 0.005

如果你只是想快速变亮，优先改这三项：toneMappingExposure、ambient、
fill。
